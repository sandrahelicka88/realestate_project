# realestate_project
Django Real Estate Website

### How to develop:

* Clone the repository
* Create virtual environment with Python3

```sh
$ python3 -m venv ./venv
```
* Activate your venv

```sh
$ source ./venv/bin/activate
```
* Install Django

```sh
$ pip install django
```
* Run app
```sh
$ python manage.py 
```
